# Deya Al Drabee Website

Next.js 14 editorial academic website for Deya Al Drabee.

## Scripts

- `npm run dev`
- `npm run build`
- `npm run start`
- `npm run lint`

## Deployment Notes

- Production metadata will use `NEXT_PUBLIC_SITE_URL` when it is set.
- On Vercel, the stable project URL is used automatically as a fallback.
- The project expects Node `20+`.
